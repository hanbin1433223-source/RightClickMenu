# macOS 界面设计参考规范

来源：[Mario Guzman's macOS Design Guidelines](https://marioaguzman.github.io/design/) + [Apple HIG](https://developer.apple.com/design/human-interface-guidelines/designing-for-macos)

---

## 设置窗口布局

| 元素 | 标准值 |
|------|--------|
| 窗口边距（左/右/下） | 20pt |
| 距标题栏/工具栏 | 14pt |
| 标签与控件间距 | 6pt |
| 分隔线上下间距 | 各 12pt |
| 底部按钮上边距 | 12pt |
| 底部按钮其他边距 | 20pt |

### 标签页窗口

| 元素 | 标准值 |
|------|--------|
| 标签页居中 | 始终居中 |
| 距标题栏 | 12pt |
| 所有边距 | 20pt |

## 侧边栏规范

| 元素 | 标准值 |
|------|--------|
| 最小宽度 | 225–275pt |
| 最大宽度 | 350–400pt |
| 层级深度 | 最多 2 层 |
| 图标着色 | macOS Tahoe+ 使用黑/白（非强调色） |

## 分组方式（按视觉重量排序）

1. **留白**（最轻）— 组间距 12–24pt，适合组少空间大的情况
2. **分隔线**（中等）— 上下各 12pt 间距，适合空间紧凑时
3. **分组框**（最重）— 内部边距 16pt，需要对称使用

## 底部按钮

- 帮助按钮：左对齐
- 取消：右侧，在确定左边
- 确定：最右边，Return 键触发
- 边距：上 12pt，其他 20pt

## 控件尺寸

- 常规控件：标签右对齐，控件左对齐，间距 6pt
- 小控件：段间距 12pt，标题到首个控件 8pt
- 极小控件：段间距 12pt，标题到首个控件 8pt，控件间距 4pt
